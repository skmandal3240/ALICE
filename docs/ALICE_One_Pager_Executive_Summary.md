# ALICE — One-Page Executive Summary

**ALICE** is a private, desktop-native AI companion built for India's knowledge economy.

### The Problem
Indian professionals lose hours daily to fragmented AI tools that forget context, speak poorly in Indian accents, and send every prompt to foreign clouds — creating DPDP Act risk for enterprises and government.

### The Solution
ALICE is a menu-bar AI orb that lives next to your cursor.  
- Push-to-talk (Ctrl+Option) with strong Indian accent support  
- Sees your full multi-monitor screen  
- Streams Claude/OpenAI responses with voice  
- Physically points at UI elements on screen  
- All API keys stay on a secure gateway (never in the app)

### Why India Wins
- Aligns with IndiaAI Mission, DPDP compliance, and data sovereignty  
- Starts with high-value macOS users (researchers, bureaucrats, founders)  
- Roadmap: Hindi + regional languages, on-device models, Windows/Linux ports

### Current State (July 2026)
- Fully working macOS app  
- DMG installer auto-built via GitHub Actions  
- Public MIT repo + production-ready CI  
- Logo and branding complete

### Business Model
Freemium (local models free) → Pro (₹999–1999/mo) → Enterprise/Government (DPDP packages + on-prem gateway).

### The Ask
₹1.5–3 Cr (grant + seed) to incorporate, hire core team, add Indian languages + on-device models, and win first enterprise/government pilots.
